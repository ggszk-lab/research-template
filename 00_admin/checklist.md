# 実行・提出チェックリスト

このチェックリストは，研究の節目（中間確認・提出前）で確認することを想定しています．

## 研究進行
- [ ] 研究目的・RQが `00_context/context.md` と一致
- [ ] 重要な判断が `00_context/decisions.md` に記録
- [ ] データ取扱いが `00_admin/data_policy.md` に準拠

## 再現性
- [ ] 実行手順（環境 / 依存 / コマンド）を記載
- [ ] 乱数シード / バージョンを記録
- [ ] 生成物（図表 / 表）が `03_results/` に整理
- [ ] クリーンな環境で再実行し，同じ結果が得られることを確認（可能な範囲で）

## 提出 / 公開
- [ ] 個人情報・機微情報の混入がない
- [ ] 図表・数値が本文と整合
- [ ] 外部共有リンクの権限確認（`00_admin/links.md`）
- [ ] 生成AI利用の有無・範囲を必要に応じて明示
