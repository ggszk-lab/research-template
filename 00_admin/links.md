# 外部リンク集

外部サービス（Drive, Notion, Issue, チケット, データカタログ等）のリンクを集約します．  
本ファイルは，プロジェクトで利用する外部サービス・参照先を「正」として管理するためのものです．

- プロジェクト管理:（Issue管理，タスクボード等）
- 共有ドライブ:（データ・資料の保管場所）
- データソース / カタログ:（外部DB，API，公開データ）
- 関連 Issue / PR:（GitHub 等）
- 参考文献 / 先行研究:（論文，資料，URL）

※ 各リンクには，必要に応じて「用途」「閲覧権限」「注意事項」を簡単に補足してください．
