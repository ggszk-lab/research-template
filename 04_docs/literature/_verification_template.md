- 目的: 投稿前の引用検証。(a) 書誌照合（著者・年・掲載先・巻号・頁・DOI）、(b) 主張精読（本文で帰属させた主張が原典と一致するか）

## 方針

- サーベイ（`/lit-survey`）とは別の活動。新規文献は探さず、既存引用の正確性のみ検証する。
- 段階1 = Web 照合（DBLP・arXiv・出版社ページ・PubMed 等）。段階2 = 原典精読（ローカル PDF・公開 HTML）。
- fetch 不可のドメインは、複数の独立な検索結果でのクロス確認に依拠する（各エントリに明記）。
- 確証が取れないものは ✅ にせず 🔍 か □ のまま残す（捏造・憶測で埋めない）。

## 凡例

- ✅ 書誌照合済・問題なし
- ⚠️ 要修正（書誌に誤り）→ 対応後は「修正済」と併記
- 🔍 要精読（帰属した主張を原典で確認する必要あり）
- □ 未照合
