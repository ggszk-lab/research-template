# Decision Log（判断・方針変更ログ）

このファイルは，研究の進行に影響する重要な判断を時系列で残すためのものです．  
詳細な議論や試行錯誤は別ドキュメントに分離し，ここでは「決定内容」を簡潔に記録します．

## 運用ルール
- 1エントリ = 1判断
- 原則 1〜5行で記述する
- 日付は決定日（YYYY-MM-DD）
- 小さな判断でも，後で解釈に影響しそうなら残す

## 記載フォーマット（推奨）

- YYYY-MM-DD: 決定内容（要約）
  - 理由:
  - 代替:（あれば）
  - 影響:（データ / 分析 / 成果物 など，必要なら）
  - 関連: Issue / PR / メモ

※ すべて埋める必要はありません．

---

## ログ

- （例）2026-01-01: 分析対象期間を YYYY-MM〜YYYY-MM に変更
  - 理由: データ欠損が多い初期期間を除外するため
  - 影響: 分析・結果
